所有的编程语言，包括 JavaScript，对浮点型数据的精确度都很难确定
function.toString()输出函数字符串而不是返回值的String
若对象的作用域包括function，而function在方法体内对它做出了改变，那么对象会被改变，而作为参数传进去的话仅仅是值传进去
通过call或者apply调用函数

两个函数第一个参数都是对象本身，apply传入的是一个参数数组，也就是将多个参数组合成为一个数组传入，而call则作为call的参数传入（从第二个参数开始）
在 JavaScript 严格模式(strict mode)下, 在调用函数时第一个参数会成为 this 的值， 即使该参数不是一个对象
在 JavaScript 非严格模式(non-strict mode)下, 如果第一个参数的值是 null 或 undefined, 它将使用全局对象替代
在 JavaScript 中，所有函数都能访问它们上一层的作用域
element.addEventListener(event, function, useCapture);
第二个参数函数对象需要通过匿名函数来传递参数，因为第二个参数只能是函数名字
